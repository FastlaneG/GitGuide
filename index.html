The Basics: Understanding Git

A simple guide teaching the git process.

Step 1. Create A Directory (Folder)

Use mkdir to create a new folder.

Step 2. Create A File

Use touch to create a file.

Step 3. Create A Git Repository

Use git init to start new repo.

Step 4. Add Files To Staging Index

Use git add * or git add 'filename' to stage files.

Step 5. Commit Changes

Use git commit -m "Your Commit Message Here" to commit your changes to repo.

Step 6. Send Your Commit Changes

Use git push origin master so you link to github.
